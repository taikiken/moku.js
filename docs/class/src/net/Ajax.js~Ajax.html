    Ajax

    fetch API を使用し Ajax request を行います

    Safari, IE はサポートしていないので polyfill ライブラリを使用します
    また、 fetch は Promise も必要としています。

    $ bower install fetch

$ bower install es6-promise

    See:

    • http://caniuse.com/#feat=fetch
    • https://github.com/github/fetch
    • https://github.com/github/fetch
    • https://developer.mozilla.org/ja/docs/Web/API/Fetch_API/Using_Fetch
    • https://developer.mozilla.org/ja/docs/Web/API/Fetch_API
    • https://developer.mozilla.org/ja/docs/Web/API/Request
    • https://developer.mozilla.org/ja/docs/Web/API/Request/Request
    • https://developer.mozilla.org/ja/docs/Web/API/Headers
    • https://developer.mozilla.org/ja/docs/Web/API/Body

    Static Public Methods

    public static option(path: string, method: string, headers: Headers | Object | null, formData: FormData | null): Request source

    fetch API へ送るオプションを作成します

    必ず設定します

    • method: GET, POST, PUT, DELETE...etc
    • cache: 'no-cache'
    • credentials: 'same-origin'

    headers, formData は存在すれば option に追加されます

    Params:

    NameTypeAttributeDescription
    path string

    Ajax request path
    method string

    GET, POST, PUT, DELETE...etc request method
    headers Headers | Object | null

    Headers option
    formData FormData | null

    送信フォームデータオプション

    Return:

    Request

    fetch API へ送る Request instance を返します

    See:

    • https://developers.google.com/web/updates/2015/03/introduction-to-fetch
    • https://developer.mozilla.org/ja/docs/Web/API/Request
