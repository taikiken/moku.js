    Ajax → AjaxThunk

    fetch API を使用し Ajax request を行います, ref: Ajax

    See:

    • http://caniuse.com/#feat=fetch
    • https://github.com/github/fetch
    • https://github.com/taylorhakes/promise-polyfill
    • https://developer.mozilla.org/ja/docs/Web/API/Fetch_API/Using_Fetch
    • https://developer.mozilla.org/ja/docs/Web/API/Fetch_API
    • https://developer.mozilla.org/ja/docs/Web/API/Request
    • https://developer.mozilla.org/ja/docs/Web/API/Request/Request
    • https://developer.mozilla.org/ja/docs/Web/API/Headers
    • https://developer.mozilla.org/ja/docs/Web/API/Body
    • https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Statements/async_function
    • http://getmesh.io/Blog/Make%20AJAX-Requests%20Great%20Again

    const ajax = new Ajax();

    Public Methods

    public start(path: string, method: string, headers: Headers, formData: FormData): Promise source

    Ajax request 開始します

    request 可能 / 不可能 flag が false の時は実行しません
    true の時は false にしリクエストを開始します

    • resolve, reject 関数確認後実行します
    • Promise instance を返します
    • json / error を返します

    Override:

    Ajax#start

    Params:

    NameTypeAttributeDescription
    path string

    Ajax request path
    method string
    • optional
    • default: GET

    GET, POST, PUT, DELETE...etc request method
    headers Headers
    • optional
    • default: null
    • nullable: true

    Headers option, token などを埋め込むのに使用します
    formData FormData
    • optional
    • default: null
    • nullable: true

    フォームデータを送信するのに使用します

    Return:

    Promise

    ajax request を開始し fetch Promise 返します
