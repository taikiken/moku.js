    Fps

    Extends:

    EventDispatcher → Polling → Fps

    フレームレート毎に UPDATE イベントを発生させます

    Example:

    // 2sec. interval
const fps = new Fps(0.5);
const update = () => {
 // code here, something do
};
fps.on(Fps.UPDATE, update);
fps.start();
