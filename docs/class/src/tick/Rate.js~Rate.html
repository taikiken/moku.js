    Rate

    Extends:

    EventDispatcher → Polling → Rate

    固定値を使用し fps を決定します

    以下のフレームレートが設定可能です

    • 60: Rate.RATE_60
    • 30: RATE_30
    • 20: RATE_20
    • 15: RATE_15
    • 12: RATE_12
    • 10: RATE_10
    • 6: RATE_6
    • 5: RATE_5

    Example:

    // 1sec interval
const rate = new Rate(Rate.Rate_60);
const update = () => {
 // code here, something do
};
rate.on(Rate.UPDATE, update);
rate.start();
