    Polling

    Extends:

    EventDispatcher → Polling

    Direct Subclass:

    Fps, Rate

    一定間隔毎に UPDATE イベントを発生させます

    Example:

    // 3sec. interval
const polling = new Polling(1000 * 3);
const update = () => {
 // code here, something do
};
polling.on(Polling.UPDATE, update);
polling.start();
