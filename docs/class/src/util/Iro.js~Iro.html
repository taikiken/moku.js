    public static hue2rgb(point: number, q: number, hue: number): number source

    HSL to RGB で saturation !== 0 な時の R, G, B 変換 helper です

    Params:

    NameTypeAttributeDescription
    point number

    (2 * l) - q
    q number

    l < 0.5 ? l * (1 + s) : (l + s) - (l * s)
    hue number

    hue

    Return:

    number

    0 ~ 1 な値を返します
